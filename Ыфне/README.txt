
Инструкция по установке:

1. Разместите все файлы из архива на вашем сервере или хостинге в одной папке.
2. Загрузите файл песни "coco_jambo.mp3" в ту же папку, где находятся файлы сайта.
3. Откройте файл index.html в браузере или настройте хостинг для автоматического открытия index.html по URL вашего сайта.
4. Нажмите кнопку "Играть 'Coco Jambo'", чтобы воспроизвести песню.

Удачи!
